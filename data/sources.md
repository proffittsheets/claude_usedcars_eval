# Data Sources Registry

All external data sources used in this project. Updated as each script is written.

---

## Vehicle Specifications

**CarQuery API**
- URL: https://www.carqueryapi.com/api/0.3/
- Script: `scripts/fetch_carquery_specs.py`
- Used for: body type, drivetrain (AWD detection), seat count, transmission type, MSRP (where available)
- Auth: None required
- Format: JSONP — strip callback wrapper before parsing
- Rate limit: ~1 request/second recommended
- License: Free for non-commercial and commercial use

---

## Fuel Economy & MPG

**FuelEconomy.gov Web Services (U.S. DOE)**
- URL: https://www.fueleconomy.gov/ws/rest/
- Script: `scripts/fetch_fuel_economy.py`
- Used for: city/highway/combined MPG, fuel type, hybrid/PHEV flag
- Auth: None required
- Format: JSON (Accept: application/json header required)
- Rate limit: None published; use reasonable delays
- License: U.S. Government data, public domain
- Key fields: `comb08` (combined MPG), `atvType` (hybrid flag), `fuelType1`

## Safety Ratings

**NHTSA 5-Star Safety Ratings API**
- URL: https://api.nhtsa.gov/SafetyRatings/
- Script: `scripts/fetch_nhtsa_safety.py`
- Used for: overall, front crash, side crash, and rollover star ratings (1–5)
- Auth: None required
- Format: JSON
- Rate limit: ~2 requests/second maximum
- License: U.S. Government data, public domain
- Two-step process: get VehicleId list → get ratings per VehicleId

**IIHS Top Safety Picks**
- URL: https://www.iihs.org/ratings/top-safety-picks
- Script: N/A — manually curated
- Used for: TSP / TSP+ award status
- Auth: None (manual curation from website)
- Data file: `data/manual/iihs_overrides.json`
- Update frequency: Annually (IIHS releases awards each spring)

## Vehicle Images

**Wikimedia Commons API**
- URL: https://commons.wikimedia.org/w/api.php
- Script: `scripts/fetch_images_wikimedia.py`
- Used for: exterior and interior photos
- Auth: None required (User-Agent header required per Wikimedia policy)
- License: Creative Commons (various — CC BY-SA 4.0 common)
- Attribution: Required — stored per image in `data/raw/images/manifest.json`
- Images stored: `data/raw/images/wikimedia/{make}/{model}/{year}/`

**Toyota Pressroom**
- Script: `scripts/fetch_images_manufacturer.py`
- Used for: official Toyota press photography (year-specific)
- URL pattern: `https://pressroom.toyota.com/vehicle/{year}-toyota-{slug}/`
- Images hosted at: `toyota-cms-media.s3.amazonaws.com`
- Coverage: 2024/2025/2026 for all Toyota models (Venza 2025/2026 falls back to Wikimedia)
- Auth: None (standard HTTP)
- License: All rights reserved — Toyota Motor North America (personal/research use)
- Terms: https://pressroom.toyota.com/terms-of-use/
- Images stored: `data/raw/images/manufacturer/Toyota/{model}/{year}/`

**Motor Trend (Hearst Autos)**
- Script: `scripts/fetch_images_manufacturer.py`
- Used for: year-specific editorial photography for Nissan, Mercedes-Benz, Lincoln
- URL pattern: `https://www.motortrend.com/cars/{brand}/{model}/{year}`
- Images hosted at: `hips.hearstapps.com/mtg-prod/` (Hearst CDN)
- Coverage: 2024/2025 for all Nissan, Mercedes-Benz, Lincoln models (~12–13 images per page)
- Auth: None (standard HTTP)
- License: All rights reserved — Hearst Autos (personal/research use)
- Terms: https://www.motortrend.com/terms/
- Images stored: `data/raw/images/manufacturer/{make}/{model}/{year}/`

**Image comparison page:** `compare_images.html` — generated by `scripts/generate_compare_images.py` to evaluate manufacturer vs. Wikimedia sources side by side.

## Pricing (MSRP)

**Manual Seed File**
- Data file: `data/manual/msrp_seed.json`
- Sources: manufacturer configurator pages (links in file)
